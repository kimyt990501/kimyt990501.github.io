# 프로젝트 개요

## 프로젝트 목적
이 프로젝트는 개인 개발 학습 블로그(kimyt990501.github.io)로, Jekyll 기반의 정적 사이트 생성기를 사용하여 GitHub Pages에 호스팅됩니다.

## 주요 정보
- **블로그 제목**: 개발공부 블로그
- **작성자**: kimyt990501 (장래희망개발자)
- **사용 테마**: Minimal Mistakes Jekyll Theme (remote theme)
- **테마 스킨**: dark
- **언어**: 한국어 (ko-KR)
- **주요 카테고리**: Java, C++, Spring

## 콘텐츠 구조
- 블로그 포스트는 `_posts` 디렉토리에 카테고리별로 분류됨:
  - `_posts/java/` - Java 학습 관련 포스트
  - `_posts/cpp/` - C++ 학습 관련 포스트
  - `_posts/spring/` - Spring 프레임워크 학습 관련 포스트

## 주요 기능
- utterances를 통한 댓글 시스템
- Google Analytics 추적 (G-4XYZGF532P)
- 태그 및 카테고리 아카이브
- 반응형 디자인
- 수학 수식 지원 (mathjax)
- 사이드바 네비게이션
